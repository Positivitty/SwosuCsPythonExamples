📚 What is a Symmetric Matrix?

A square matrix is symmetric if it equals its transpose. In other words, element at position (i,j) equals element at position (j,i) for all valid i,j.

Mathematical Definition: A matrix A is symmetric if A = AT, where AT is the transpose of A.

Visual Check: The matrix should be identical when reflected across its main diagonal (top-left to bottom-right).
